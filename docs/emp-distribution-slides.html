What is EMP Distribution?

Single package — everything to run an EMP board

Full Image New board → flash & go .img.gz • ~2 GB
Software Release Existing board → update over SSH .tar.gz • ~80 MB

Same software content, different delivery mechanism

Software Update

$ ./deploy.sh emp-board-01 emp-release-26.04.1.tar.gz
  • One command — deploys over SSH
  • Atomic symlink swap — zero-downtime preparation
  • Previous version kept for instant rollback
/opt/emp/
  current → releases/26.04.1/ # atomic symlink
  releases/
    26.04.1/ # previous — kept
    26.04.1/ # new — active
  version

Rollback

$ ./rollback.sh emp-board-01
  • Swaps symlink back to previous release
  • Restarts all services
  • Always safe — previous version untouched on disk
v26.04.1problem
→
rollback.sh
→
v26.04.1restored

No rebuild. No re-download. Symlink swap + service restart.
